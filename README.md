# Tic Tac Toe
Setting up a game of Tic Tac Toe using the `Vite` development tooling.

## Play the game
To [play this game](https://ebabel-eu.github.io/tic-tac-toe-vite/) is 100% free, and so is the source code.

## Install
`npm i`

## Start locally for dev
`npm run dev`
